Employee Performance Evaluation System

A full-stack web application designed to help organizations evaluate employee performance, set goals, provide feedback, and generate performance analytics.
The system ensures secure workflows, role-based access, data validation, and robust error handling for Managers, Employees, and Admins.

 Project Objective

To build a scalable and reliable performance management system that enables:

Transparent performance evaluation

Continuous feedback

Goal tracking

Actionable analytics

Secure role-based access

 User Roles & Functionalities
 Manager

Evaluate employee performance using structured scorecards

Set performance goals and KPIs

Provide continuous and role-specific feedback

Review and approve employee self-evaluations

Generate and view performance reports

Monitor team progress through dashboards

 Employee

Submit self-evaluation forms

Track assigned goals and performance scores

View manager feedback and evaluation history

Update personal goals and skill progress

Access performance trends over time
